소개
--------------
LED와 스위치를 이용해서 간단하게 만드는 크리스마스 카드

납땜도 아두이노도 사용하지 않고도 3D프린터를 이용해 멋진 크리스마스 카드를 만들어 보아요. 전도성 실을 이용해서 쉽고 간단하게 크리스마스 카드를 만들어 보도록 합시다!
